# Lab Seminar Series

This repository tracks the seminars of the [Your Lab Name].

## How to add a seminar

1. Create a new folder in `seminars/` named:
   `YYYY-MM-DD-first-last/`
2. Copy the templates from `templates/`.
3. Fill in:
   - `metadata.yaml`
   - `abstract.md`
4. Upload slides as `slides.pdf`.
5. Update `schedule.md`.
6. Commit your changes.
